nextrs

File-based routing for Rust. Each folder under app/ is a route segment; convention files page.{rs,html}, layout.{rs,html}, and loading.{rs,html} compose the response.

Examples

1. Just a page
A minimal route — only page.rs + page.html. Renders synchronously.
app/simple/{page.rs, page.html}
2. Page with loading
Adds loading.{rs,html}. The framework streams the loading shell first, then swaps in the page when it's ready.
app/with-loading/{page.rs, page.html, loading.rs, loading.html}
3. Page with layout and loading
Adds layout.{rs,html} on top of the previous example. The layout wraps just this section, on top of the root layout.
app/with-layout/{layout.rs, layout.html, page.rs, page.html, loading.rs, loading.html}
