3. Page with layout and loading

The full set of conventions. The root layout (nav bar) still wraps everything. On top of that, this route's layout.html adds the sidebar to the left. The loading.html renders inside the sidebar layout while page.rs sleeps 800ms.

Files
  • app/with-layout/layout.rs — renders the sidebar wrapper around child content
  • app/with-layout/layout.html — sidebar template; uses {% raw %}{{ children|safe }}{% endraw %}
  • app/with-layout/page.rs — sleeps 800ms then renders
  • app/with-layout/page.html — this template
  • app/with-layout/loading.rs — renders the loading shell
  • app/with-layout/loading.html — spinner shown inside the sidebar while the page resolves
